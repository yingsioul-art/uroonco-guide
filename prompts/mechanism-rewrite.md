# 藥理機轉改寫 Prompt — 護理師衛教導向

> 用途:把原廠仿單／教科書語氣的「作用機轉」原文,改寫成護理師能夠理解、
> 並能直接轉述衛教給病人的版本。
> 適用於單株抗體類藥品(免疫檢查點抑制劑 / 標靶 / ADC)。

---

## 使用方式

複製下方整段 Prompt,把 `[藥品名稱]` 換成藥名(例如 Avelumab、Pembrolizumab),
把 `[貼上原文]` 換成原文機轉說明,送給語言模型即可。

---

## Prompt 本體

```
以下是 [藥品名稱] 的藥理機轉原文說明:

「[貼上原文]」

請依照下列原則改寫,說明的受詞是護理師,目的是幫助護理師理解後能對病人進行衛教指導:

1. 說明此藥為單株抗體(monoclonal antibody, mAb),並明確標註其分類屬於下列哪一種,且附上英文全名:
   - 免疫檢查點抑制劑(immune checkpoint inhibitor, ICI)
   - 標靶藥品(targeted therapy)
   - 抗體結合藥物(antibody-drug conjugate, ADC)

2. 所有專有名詞保留中文,並在第一次出現時加上英文全名與縮寫,例如:
   程序性死亡配體 1(programmed death-ligand 1, PD-L1)

3. 艱深的免疫或分子機轉用詞,降階改用護理師日常工作能理解的比喻或白話說明,例如:
   - 受體結合 → 黏住 / 接上
   - 抑制訊號 → 讓...停止攻擊 / 發出「不要攻擊」的訊號
   - 腫瘤免疫逃脫 → 癌細胞偽裝成正常組織

4. 輸出格式請分為兩個層次,以標題區隔:

   【簡要說明】
   - 2~3 句話
   - 說明藥品分類、作用對象、核心機轉
   - 使用最白話的語言,讓護理師能直接口頭轉述給病人

   【詳細說明】
   - 連貫的段落,勿用條列
   - 依序涵蓋:藥品分類 → 癌細胞如何逃脫免疫 → 藥物如何介入 → 最終效果
   - 專有名詞第一次出現須附英文全名與縮寫
   - 結尾不需要加護理指導摘要
```

## 改寫前後對照範例(以 Avelumab 為例)

### 原文(典型仿單寫法)

> Avelumab 是一種人類免疫球蛋白 G1 (IgG1) 單株抗體,可直接與程序性死亡配體
> 1 (PD-L1) 結合。藉由阻斷 PD-L1 與程序性死亡受體 1 (PD-1) 以及 B7.1 受體
> 之間的交互作用,Avelumab 可移除 PD-L1 對細胞毒性 T 細胞 (CD8+) 的抑制作用,
> 進而恢復抗腫瘤免疫反應。

### 改寫後(預期產出格式)

【簡要說明】
Avelumab 是一種免疫檢查點抑制劑(immune checkpoint inhibitor, ICI),屬於
單株抗體(monoclonal antibody, mAb)。它的工作是把癌細胞身上的「偽裝外衣」
拆掉,讓我們自己的免疫細胞能再次認出癌細胞並攻擊它。

【詳細說明】
Avelumab 屬於單株抗體(monoclonal antibody, mAb)中的免疫檢查點抑制劑
(immune checkpoint inhibitor, ICI)。正常情況下,T 細胞(T cell,人體
最主要的抗癌免疫細胞)會巡邏全身,看到不正常的細胞就會發動攻擊。但有些
癌細胞會在表面長出一個叫做程序性死亡配體 1(programmed death-ligand 1,
PD-L1)的「偽裝牌」,當 T 細胞身上的程序性死亡受體 1(programmed death-1,
PD-1)接觸到這塊牌子時,就會收到「不要攻擊」的訊號,T 細胞便誤以為對方
是好人而放過癌細胞——這就是癌細胞的偽裝逃脫機制。Avelumab 的角色就像是
搶先一步把這塊偽裝牌黏住、蓋掉,讓 T 細胞接觸不到 PD-L1,也就收不到「不要
攻擊」的訊號,於是 T 細胞重新認出癌細胞並發動毒殺,把腫瘤打下來。

## 補充提示(可選)

若需要同時產出 **病人版口語腳本**,可在 Prompt 結尾加上:

> 5. 額外提供「病人版口語腳本」一段(3~4 句),用「您」稱呼病人,
>    完全不出現英文縮寫,只用最簡單的比喻。

若需要 **針對特定癌別** 微調舉例,可在 Prompt 開頭補一句:

> 此藥目前用於 [癌別],請在比喻時可貼近該癌別的衛教情境。
